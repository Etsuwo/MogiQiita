# MogiQiita

## pod
クローンの後，ライブラリをインストールをして下さい．
``` 
pod install 
```

## .env
実行には.envファイルが必要です．  
MogiQiita/以下にある.env.sampleを.envに置き換えて下さい．  
MogiQiitaApp.swiftで読み込んでいるので，置き換えたらそのまま実行して大丈夫です．  
.envファイルの取得については僕までご連絡ください．
